# Command Center

Your personal kanban board. Use `/board` to view, `/board move <id> <status>` to organize.

## Quick Commands

| Command | Action |
|---------|--------|
| `/idea <text>` | Capture a new idea |
| `/board` | Show kanban view |
| `/board move <id> doing` | Start working on item |
| `/tasks` | List items in progress |
| `/tasks done <id>` | Complete an item |

---

## How It Works

1. **Capture** - Use `/idea` to quickly jot things down
2. **Organize** - Use `/board` to see everything and prioritize
3. **Execute** - Use `/tasks start <id>` to begin, `/tasks done <id>` to complete
4. **Review** - Periodically archive done items with `/board archive`

Items flow: **Backlog → Doing → Done → Archived**
