# Solutions

Specific technical problems you've debugged and fixed.

## When to Add Here

- ✅ Spent > 30 minutes debugging something
- ✅ Found a non-obvious solution
- ✅ Hit an issue that could happen again
- ✅ Solution isn't obvious from documentation

## File Naming

Use descriptive, searchable names:
- `auth-token-refresh-race-condition.md`
- `cors-preflight-405-error.md`
- `prisma-connection-pool-exhaustion.md`
- `react-hydration-mismatch-dates.md`

## Template

```markdown
# [Problem Title]

## The Problem
[Error messages, symptoms, what was happening]

## The Cause
[Root cause - why it was happening]

## The Solution
[Code, config, or steps that fixed it]

## Keywords
[error messages, library names, symptoms for searching]

## Metadata
- Captured: YYYY-MM-DD
- Projects: [which projects had this]
- Libraries: [relevant library versions]
```

## Tips

- Include exact error messages (makes searching easier)
- Include the "wrong" approaches you tried first
- Note which projects this applies to
- Tag with library versions (solutions can become outdated)
