## Adding New Projects

When creating a new project:

1. Create the project directory in `c:\Users\GGPC\CascadeProjects\`
2. Initialize git: `git init`
3. Create `.claude/settings.local.json` for project-specific Claude Code settings
4. Create `.claude/pm/` directory for project management files
5. Add entry to this registry
